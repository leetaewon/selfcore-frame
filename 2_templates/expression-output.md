# STEP 7 결과물 출력 형식

Claude는 STEP 7 완료 시 아래 형식으로 웹사이트 표현 세트를 출력한다.
사용자의 이름과 확정된 내용으로 [대괄호] 부분을 채운다.

---

## [이름]의 웹사이트 표현 세트

| 요소 | 내용 |
|---|---|
| **Role** | [직함 1] · [직함 2] · [직함 3] |
| **Identity** | [나를 설명하는 한 문장] |
| **Slogan** | [행동 방침 한 줄] |
| **Message** | [방문자에게 건네는 첫 마디] |

**서브카피**
[1단락 — 방문자의 상황·감정]
[2단락 — 이 사람만의 방식 + 대상 명시]

### 각 요소의 출처

| 요소 | 출처 |
|---|---|
| Role | What을 바탕으로 — 내가 만드는 것을 직함으로 표현 |
| Identity | 미션 선언문을 방문자 관점으로 전환 |
| Slogan | How를 압축 — 내가 다르게 하는 방식을 한 줄로 |
| Message | Why를 방문자 언어로 전환 — 방문자의 상황을 먼저 인정 |
| 서브카피 | Message를 받쳐주는 2단락 — 상황 공감(1단락) + 차별화·대상 명시(2단락) |

---

## 작성 원칙

- Role: 3개의 단어·구, 중간점(·)으로 연결, 20자 이내
- Identity: "~를 위한 ~" 형태, 20자 내외
- Slogan: 동사형으로 끝나는 것이 좋음, 10자 내외
- Message: 방문자의 감정·상황을 먼저 인정하는 문장
- 서브카피: 각 단락 1~2문장, 전체 4문장 이내. Message와 같은 언어 결 유지
- 다섯 가지를 함께 읽었을 때 한 사람의 브랜드로 읽혀야 한다

---

## 예시 — 이태원쌤의 웹사이트 표현 세트

| 요소 | 내용 |
|---|---|
| **Role** | AI 실천가 · 워크플로우 코치 · 디지털 멘토 |
| **Identity** | AI로 일하는 방식을 바꾸는 1인기업 전문가를 위한 디지털 코치 |
| **Slogan** | 먼저 해보고, 함께 만듭니다 |
| **Message** | AI 앞에서 혼자 헤매지 않아도 됩니다 |

**서브카피**
자기 일에 AI를 연결하고 싶은데 어디서부터 시작해야 할지 모르겠나요? 그 막막함을 저도 겪었습니다.

이태원쌤은 도구 소개가 아닌 실제 워크플로우를, 내가 겪은 과정 그대로 보여줍니다. 강사·코치·컨설턴트 — 이미 전문성이 있는 분들이 AI를 자기 일에 녹일 수 있도록 길을 내고 함께 갑니다.

### 각 요소의 출처

| 요소 | 출처 |
|---|---|
| Role | What(AI 활용 강의·콘텐츠) → AI 실천가·코치·멘토로 표현 |
| Identity | 미션 선언문의 X(1인기업 전문가)와 Y(AI로 일하는 방식을 바꾼다)를 방문자 관점으로 전환 |
| Slogan | How(먼저 써보고 검증, 과정 그대로 보여준다) → "먼저 해보고, 함께 만듭니다"로 압축 |
| Message | Why(혼자 막히게 두고 싶지 않다) → "혼자 헤매지 않아도 됩니다"로 방문자 언어 전환 |
| 서브카피 | Why의 계기(막막함 공감) + How·X(실제 워크플로우, 강사·코치·컨설턴트 대상)를 풀어서 설명 |
